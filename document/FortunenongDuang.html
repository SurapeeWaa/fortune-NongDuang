  1. วัตถุประสงค์ (Purpose)

ระบบ Fortune-nongduang Chat Bot จะช่วยในการทำนายดวงชะตาของแต่ละคน และการทำนายความฝัน โดยนอกจากนี้จะทำนายเลขเด็ดของแต่ละดวงไว้ด้วย แต่ทั้งนี้ทั้งนั้นขึ้นอยู่กับวิจารณญาณส่วนบุคคล

  1. ขอบเขต (Scope)

        คนที่มีความเชื่อเรื่องดวงชะตาและการทำนายฝัน ระบบจะทำการตอบโต้ข้อความผ่านทาง LINE API

  1. บทนิยามของคำศัพท์ (Definitions)

คำศัพท์

ความหมาย

Chat bot

ระบบตอบโต้ข้อความอัตโนมัติ

LINE

หน่วยงานให้บริการด้านการสื่อสารระหว่างโทรศัพท์ไร้สายผ่านแอพลิเคชั่น

Application Programming Interface หรือ API

ช่องทางที่จะเชื่อมต่อกับผู้ให้บริการ

IEE

สถาบันวิศวกร ทำหน้าที่ดูแลมาตรฐานวิจัยเกี่ยวกับไฟฟ้าและคอมพิวเตอร์

Sticker

  1. คุณลักษณะของระบบซอฟต์แวร์ (Software system attributes)
